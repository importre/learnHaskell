# learnHaskell
하스켈 학습

## 설치
Stack 설치
```
brew install haskell-stack
```

## 프로젝트 생성, 빌드, 실행
```
stack new my-project
cd my-project
stack setup
stack build
stack exec my-project-exe
```

## REPL 실행
```
stack ghci
```
